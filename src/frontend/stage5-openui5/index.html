Stage 5 — OpenUI5 ODataModel + Data Binding
Compare to Stage 2 and Stage 3: There is no fetch() call in this page. The ODataModel reads the service metadata, then fetches data automatically when the table binding is activated. The filter bar assembles $filter parameters; column sort assembles $orderby; the "More" button uses $skip. Open DevTools (F12 → Network tab) to see the exact URLs being fired.
